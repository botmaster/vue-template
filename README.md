# Vue-template

## Usage

```
npm install -g vue-cli
vue init la-haute-societe/vue-template my-project
cd my-project
npm install
npm run dev
```

## What's Included

 - LESS
 - Spritesmith
 - GSAP
 - grunt-ssh-deploy-release
 - GitLab CI
